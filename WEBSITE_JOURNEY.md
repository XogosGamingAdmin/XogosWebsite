1. **Browse Content** - View all public pages freely
2. **Read Blog Posts** - Access 339+ educational articles
3. **View Board Info** - See board members and initiatives (no financials)
4. **Subscribe to Newsletter** - Enter email on homepage or blog
5. **Sign In** - Access board member portal (if authorized)

### Newsletter Subscription Flow

1. User visits Homepage (`/`) or Blog (`/blog`)
2. Enters email in newsletter form
3. API endpoint `/api/newsletter` processes subscription
4. Email stored in `newsletter_subscriptions` database table
5. Confirmation displayed to user
